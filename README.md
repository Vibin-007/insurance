# Insurance Cost Prediction using Linear Regression

## Project Overview
This project uses a Linear Regression Machine Learning model to predict insurance charges based on customer attributes such as age, BMI, smoking habits, number of children, gender, and region.

The workflow includes:
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Feature encoding
- Model training
- Performance evaluation
- Prediction generation

---

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

---

## Features
- Data cleaning and preprocessing
- Correlation analysis
- Visualization using charts and plots
- Linear Regression model implementation
- Model accuracy evaluation
- Real-world insurance cost prediction

---

## Machine Learning Workflow
1. Import dataset
2. Perform data preprocessing
3. Encode categorical features
4. Split dataset into training and testing sets
5. Train Linear Regression model
6. Evaluate model performance
7. Predict insurance charges

---

## Evaluation Metrics
The following metrics are used:
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score
